Color is the primary signal

Glass tint, text, caret, and pillars all derive from one continuous interpolation: blue (slow) → green (ideal, 140 wpm) → red (fast). Eased with t1.6 so small deviations stay close to green.

Caret, not slider thumb

A 4×5pt triangle below the spectrum line, pointing up. Reads as a meter readout. It moves smoothly with WPM; it is not interactive.

Bars trim, never pad

Each row's pillars are exactly the count, left-aligned. No empty trailing slots. Width naturally cascades by frequency.

Top WPM and state-row text are unchanged · spectrum line acts as middle separator · caret pointer triangles up at current pace · TOO SLOW / IDEAL / TOO FAST flips at 115 / 175 wpm boundaries · gradual color fade tied to deviation from ideal pace · widget hidden when mic is off.
