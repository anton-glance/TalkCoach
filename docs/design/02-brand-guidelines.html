00 · Scope & precedence

What this doc is and isn't

This is the Locto brand reference adapted for the v1 project (Session 018). The product is named Locto for user-facing surfaces; the repository, Xcode scheme, and bundle identifier (com.talkcoach.app) retain the working name TalkCoach for v1.

This doc wins on: visual identity (colors, type, mark/wordmark, motion principles), voice/tone for user-facing copy, brand component visual specs.

This doc does NOT win on:

  • Content scope — 02_PRODUCT_SPEC.md wins. Filler-stroke components below are deferred to v2.0; v1 widgets do not show fillers.
  • Technical architecture — 03_ARCHITECTURE.md wins. Persistence is SwiftData (not GRDB as the source Locto package suggested); transcription is Architecture Y (Apple SpeechAnalyzer + Parakeet via Core ML for Russian).
  • Widget shell decisions — 03_ARCHITECTURE.md §8 wins. Widget size is 144 × 144 pt (not 320 × 320 pt). Material is Liquid Glass with state ink tint and hover saturation shift (not solid pastel gradients). Widget visibility is persistent during mic-active session with "Listening…" placeholder (not show-on-first-token).

When this doc and a higher-precedence doc conflict, the higher-precedence doc wins. The Locto reference's source intent — calm, on-device, brand-teal-and-state-inks — is preserved; specifics that contradict our locked product or architecture are overridden inline below or in docs/design/01-design-spec.md.

01 · Logo

The mark and the wordmark

A single ring with a centered dot. The ring is the listening boundary — ambient awareness, contained on-device. The dot is the moment of speech. The mark and the product UI are deliberately two design languages: the mark is the brand, the UI is the function.

Sizes and clear space

Minimum sizes prevent the mark from becoming a blob. Clear space prevents other elements from crowding it. Below the listed minimum, use the menu bar version (no inner ring detail).

16 px · favicon, menu bar
32 px · UI controls

Don'ts

×
Don't stretch, skew, or rotate the mark — proportions are fixed.
×
Don't apply drop shadows, glows, or AI-style gradients to the mark.
×
Don't recolor the mark in state colors (blue / green / coral). Brand only.
×
Don't place the mark on busy or low-contrast backgrounds without the squircle bg.
02 · Color

Brand color, state colors, neutrals

Two parallel palettes. The brand color is teal — used in logo, web, App Store hero, social. The state palette (blue / green / coral) is for in-product feedback only. Never use a state color for brand identity, and never use the brand color to imply a feedback state.

03 · Typography

One font, used carefully

Inter, regular and medium weights, with a single light weight reserved for hero numbers. Tabular figures everywhere a number appears. Tight tracking on display sizes, neutral tracking on body. Nothing else.

Type stack

Inter by Rasmus Andersson. Open source, weights 300 / 400 / 500 / 600 used. Available from Google Fonts. Stylistic set ss01 and contextual alternates cv11 are on by default. Tabular figures (tnum) on for any numerical display.

05 · Voice

How Locto talks

Direct, present, calm. Use specific verbs, not coaching jargon. Address the user in second person. Praise what was on point before flagging what wasn't.

Do

You held 161 wpm across 22 minutes.

Don't

Your speech velocity demonstrated strong consistency metrics.

Do

"like" is your tell.

Don't

Filler-word frequency analysis indicates elevated occurrence patterns.

Do

Pace was on point.

Don't

Excellent! Your pace was crushing it!
